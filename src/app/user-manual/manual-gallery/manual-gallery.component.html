Gallery

The Gallery is a powerful tool that allows you visualize your journeys in a more engaging way. It provides a user-friendly interface for viewing and managing your journeys. The Gallery is accessible in the left navigation bar with name Gallery.

Please find the detailed video tutorial on the Gallery in the following link:

Note:
  • Any authenticated user can access the Gallery including "Guest" user.
  • Gallery displays only Journeys that are published, Journeys that are not published yet and in Draft status will not be displayed in the Gallery.
  • Gallery displays all Journeys that are created by the authenticated user and shared with the authenticated user.
