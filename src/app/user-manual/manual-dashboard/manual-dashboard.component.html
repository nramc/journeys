Dashboard

The Dashboard is a powerful tool that allows you visualize your journeys in a more engaging way. The Dashboard is accessible in the left navigation bar with name Dashboard.

Dashboard provides two main functionalities:

  • Visualize all your journeys in Map View
  • Visualize all your journeys in Calendar View

Please find the detailed video tutorial on the Dashboard in the following link:

Note:
  • Any authenticated user can access the Dashboard including "Guest" user.
  • Dashboard displays only Journeys that are published, Journeys that are not published yet and in Draft status will not be displayed in the Dashboard.
  • Dashboard displays all Journeys that are created by the authenticated user and shared with the authenticated user.
