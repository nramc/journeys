User Onboarding

User Onboarding is a crucial step in ensuring that new users can effectively navigate and utilize the platform. The onboarding process is designed to be user-friendly and intuitive, allowing users to quickly familiarize themselves with the platform's functionalities.

The onboarding process consists of several key steps that guide users through the initial setup and configuration of their accounts. These steps include:

  • Create Account: Users can create an account by providing their email address and setting a password.
  • Activate Account: After creating an account, users will receive an activation email. They must click the link in the email to activate their account.
  • Login: Once the account is activated, users can log in using their email and password.
  • Setting up a profile: After logging in, users are prompted to set up their profile by providing additional information.
