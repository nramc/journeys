Tools

The tools section provides a comprehensive overview of the various tools available within the Journey platform. Each tool is designed to enhance your experience and streamline your workflow. You can find link to Tools in left side navigation menu.

At the moment, Image size reducer is the only tool available in Journey. This tool allows you to reduce the size of images without compromising their quality. This is particularly useful for optimizing images before uploading them to the platform.

Note: Tools are available to all users.
