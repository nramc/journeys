Getting Started

To get started with Journey, We recommend you to explore the Journey application as Guest user and familiarize yourself with the features and functionalities.

You can explore the application without creating an account. This allows you to test the application and see if it meets your needs before signing up.

Once are you are comfortable with the Journey application, You can create a free account to unlock additional features and save your journeys.

To explore Journey as a guest user, you can navigate to the Login page and click Explore as Guest.
