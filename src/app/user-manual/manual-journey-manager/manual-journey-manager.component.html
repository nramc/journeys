Journey Manager

The Journey Manager is a powerful tool that allows you to manage your journeys effectively.

  • The Journey Manager is accessible in the left navigation bar with name Journey.
  • It provides a user-friendly interface for creating, editing, and deleting journeys.
  • It allows you to view detailed information about each journey, including its photo, videos, geolocation, narration and other relevant data.
  • It enables you to track the status and progress of your journeys easily.
  • It provides a comprehensive overview of all your journeys, allowing you to manage them efficiently.

Note:
  • Any authenticated user can access the Journey Manager including "Guest" user.
  • Dashboard displays all Journeys that are created by the authenticated user and shared with the authenticated user.
